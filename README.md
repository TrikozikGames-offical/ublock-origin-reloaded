# uBlock Origin Reloaded

**uBlock Origin Reloaded** is an updated version of the popular ad blocker uBlock Origin, adapted for Google Chrome with Manifest V3 support.  

## Features
- Full support for Manifest V3.
- Background script rewritten as a service worker.
- Modern network request blocking using declarativeNetRequest.
- Updated permissions and host_permissions.
- Fully compliant with CSP V3 (no inline scripts).
- Retains original filters and blocking logic.
